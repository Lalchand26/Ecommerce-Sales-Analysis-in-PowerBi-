# Ecommerce-Sales-Analysis-in-PowerBi-
This project presents an interactive E-Commerce Sales Analysis Dashboard built using Power BI. The dashboard helps analyze sales performance, profit trends, customer behavior, and product category performance through dynamic visualizations 
# Sales Dashboard - Power BI

## Tools Used
- Power BI
- MySQL
- Excel

## KPIs
- Total Sales
- Total Profit
- Previous Month Sales
- Growth %
